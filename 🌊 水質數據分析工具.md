# 🌊 水質數據分析工具

一個完整的線上水質監測數據分析系統，支持 CSV 數據導入、標準格式圖表生成和統計分析。

## 功能特性

### 📊 1. 數據導入
- 支持 CSV 格式數據導入
- 自動數據驗證和分類
- 實時數據預覽

### 📈 2. 圖表生成
- **標準格式圖表**：按分類（地面水、滲出水、地下水）分別顯示
- **監測點排序**：按 MW-01, DW-02, MW-03, DW-04... 順序排列
- **五大指標支持**：
  - 水溫 (℃)
  - pH 值
  - 導電度 (μmho/cm)
  - 溶氧 (mg/L)
  - 濁度 (NTU)
- **靈活篩選**：按日期範圍、監測點、分類篩選

### 📄 3. 統計分析
- 自動計算各項指標的統計量（平均值、最小值、最大值、標準差）
- 按分類和監測點的詳細統計
- 生成文本報告

## 使用指南

### 第 1 步：準備 CSV 數據

CSV 文件應包含以下列（順序可不同）：
```
採樣日期,監測點,分類,水溫(℃),pH值,導電度(μmho/cm),溶氧(mg/L),濁度(NTU)
2026-01-22,MW-01,地下水,24.8,7.73,447,0.55,421
2026-01-22,DW-02,地下水,25.1,7.82,445,0.48,410
2026-01-22,上池,地面水,22.1,7.41,147.3,7.28,2.5
```

**日期格式支持**：
- `YYYY-MM-DD`（推薦）
- `YYYY/MM/DD`
- `MM/DD/YYYY`

### 第 2 步：導入數據

1. 點擊「📊 1. 導入數據」標籤
2. 點擊「選擇 CSV 檔案」選擇您的數據文件
3. 點擊「📤 導入數據」按鈕
4. 等待導入完成，查看數據預覽

### 第 3 步：生成圖表

1. 點擊「📈 2. 生成圖表」標籤
2. 設置日期範圍（默認最近 90 天）
3. 選擇要分析的監測點（可多選）
4. 選擇要分析的分類（可多選）
5. 選擇指標（水溫、pH、導電度、溶氧、濁度）
6. 點擊「📊 生成標準格式圖表」

### 第 4 步：查看統計報告

1. 點擊「📄 3. 數據統計」標籤
2. 點擊「📊 生成統計報告」
3. 查看各項指標的統計數據

## 常見問題

### Q: 如何新增新的監測點？
A: 在 CSV 文件中添加新的監測點名稱和數據，導入時會自動識別。

### Q: 支持多少條數據？
A: 理論上支持無限條，但建議不超過 10,000 條以保持性能。

### Q: 可以導出圖表嗎？
A: 可以。右鍵點擊圖表，選擇「另存圖片」即可導出為 PNG。

### Q: 數據是否會上傳到服務器？
A: 不會。所有數據處理都在本地瀏覽器中進行，完全隱私保護。

### Q: 如何修改圖表顏色？
A: 編輯 `data.js` 中的 `COLORS` 對象，修改對應監測點的顏色代碼。

## 自定義

### 修改監測點順序
編輯 `data.js` 中的 `SITE_ORDER` 數組：
```javascript
const SITE_ORDER = [
    "MW-01", "DW-02", "MW-03", "DW-04", "MW-05",
    "MW-06", "MW-07", "MW-08", "MW-09", "DW-10"
];
```

### 修改顏色方案
編輯 `data.js` 中的 `COLORS` 對象：
```javascript
const COLORS = {
    "MW-01": "#1f77b4",  // 修改為您想要的顏色
    // ...
};
```

### 添加新指標
1. 在 CSV 中添加新列
2. 在 `data.js` 中的 `getMetricName()` 函數添加映射
3. 在 `index.html` 中的 `<select>` 添加新選項
